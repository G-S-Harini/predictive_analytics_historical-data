# Predictive Analytics Using Historical Data

## Objective
Build a predictive model to forecast future sales trends using historical supermarket sales data.

## Tools Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Google Colab

## Dataset Features
- Unit Price
- Quantity
- Total Sales
- Tax
- Gross Income
- Rating

## Methodology
1. Loaded and cleaned the dataset.
2. Selected Unit Price and Quantity as input features.
3. Used Linear Regression for prediction.
4. Evaluated model accuracy using R² Score and MAE.
5. Visualized Actual vs Predicted Sales.
